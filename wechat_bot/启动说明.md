# 微信 AI 顾问 Bot 启动说明

## 前提条件
- 微信 PC 版已打开并登录（必须是桌面版，不是网页版）
- Python 3.8+ 已安装（在 Windows 上，不是 WSL）

## 安装依赖

在 Windows PowerShell 中运行（注意是 Windows 的 Python，不是 WSL）：

```powershell
cd "C:\Users\gumayusi\Desktop\陈姐烘焙\wechat_bot"
pip install -r requirements.txt
```

## 配置要监听的联系人

打开 bot.py，修改这一行：

```python
LISTEN_TARGETS = [
    "文件传输助手",   # 先用这个测试，给自己发消息
    # "陈老板",       # 改成真实客户的微信昵称（与微信里显示完全一致）
]
```

## 启动 Bot

```powershell
python bot.py
```

看到 `✅ 已连接` 和 `🚀 Bot 已启动` 后，去微信给「文件传输助手」发一条消息，Bot 会自动回复。

## 测试流程

1. 启动 bot.py
2. 在微信里给「文件传输助手」发：`我刚开了一家烘焙店，完全不知道怎么推广`
3. 几秒内会自动回复

## 注意事项

- 微信桌面端需要保持打开（不能最小化到系统托盘后自动休眠）
- 微信版本更新后可能需要更新 wxauto（`pip install wxauto --upgrade`）
- WxAuto 不支持 Mac 版微信
