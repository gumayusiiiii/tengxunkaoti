# 预算策略 & 店铺阶段策略
# Agent 根据用户预算和店铺阶段，从此文件中选取对应的策略框架

---

## 一、店铺生命周期阶段策略

### 阶段一：冷启动期（开店 0-3 个月）

**核心目标：让附近的人知道有这家店**

这个阶段最常见的错误是：一上来就想做"促销"或"折扣"。
折扣吸引来的是价格敏感的人，不是真正喜欢你的人。这个阶段要做的只有一件事：**让对的人发现你**。

**必做策略清单：**
```
① 地图认领（高德+百度）
   → 让"搜附近烘焙"的人能找到你
   → 费用：免费
   → 时间成本：1小时完成

② 小红书本地内容（发10-15篇建立基础）
   → 主题：带地名的产品笔记 + 开店日记
   → 格式：产品特写图 × 3-5 + 地址/营业时间
   → 费用：免费
   → 发布节奏：每周 2-3 篇，持续 1 个月

③ 大众点评认领 + 邀请前20个真实评价
   → 主动请进店顾客帮写评价（真诚请求，不要给钱）
   → 费用：免费

④ 开业期间做一次"限时体验活动"（只做一次，不要长期打折）
   → 例：开业前两周，每天限量10份新品9折，让顾客"有理由来试试"
   → 活动结束后，恢复原价，不要形成"这家店总打折"的印象
```

**这个阶段不要做的：**
- 不要花大钱做朋友圈广告（你还不知道什么内容有效果）
- 不要做团购（团购客人价格敏感，复购率极低）
- 不要同时试所有平台（分散精力，等于每个平台都做不好）

---

### 阶段二：成长期（开店 3-12 个月）

**核心目标：建立口碑 + 把新客变成回头客**

这个阶段知名度已经有一些基础，新问题是：来了一次的人，怎么让他们再来？

**必做策略清单：**
```
① 建立微信私域（开始认真维护顾客关系）
   → 每个来过的顾客，扫码进微信群
   → 群内每周发 3 次内容（新品/日常/限量通知）
   → 费用：免费，但需要每周 30 分钟维护

② 小红书从"发内容"升级到"优化爆款"
   → 分析哪篇内容互动最好，复制这种类型的内容
   → 选表现最好的 1 篇，花 100-200 元薯条加热测试
   → 建立"每月至少 1 篇有效果的笔记"的节奏

③ 大众点评霸王餐 / 探店邀请
   → 邀请本地探店博主（3000-1万粉丝的小博主效果最好）
   → 费用：免费产品体验（成本约 50-100 元/次），不需要付钱
   → 目标：每月积累 3-5 条图文评价

④ 设计"第二次来的理由"
   → 例：「今天买了就给你一张下次用的卡，下次来换任意一款小点心」
   → 不是折扣，而是让人感觉"下次再来"是一件自然的事
```

**这个阶段开始可以试验付费：**
- 预算 1000 元/月以内：先全部用在小红书薯条（测试哪种内容有效）
- 预算 1000-3000 元/月：薯条 + 微信朋友圈广告（定向周边 3 公里）

---

### 阶段三：稳定期（开店 1 年以上）

**核心目标：维持复购 + 季节性拉新**

这个阶段最大的危机是：老顾客慢慢流失，但新顾客没有补充进来。

**必做策略清单：**
```
① 季节性/节日活动（每季度至少一次）
   → 元旦/春节/情人节/母亲节/中秋/圣诞 — 选 3-4 个最相关的
   → 提前 2 周在小红书预热，提前 1 周在微信群预告
   → 配合限量 / 预订机制，制造稀缺感

② 会员/积分体系（简单版）
   → 不需要系统，纸质集章卡就够
   → 例：集满 10 个章，换一个你最受欢迎的产品
   → 目的：给老顾客一个"下次来"的具体理由

③ 老带新激励（持续运营）
   → 微信群固定每月一次"带朋友来"活动
   → 例：「你带一个新朋友来，你们各享 8.8 折，新朋友还送欢迎小礼物」

④ 内容升级：从"发产品"到"建立人设"
   → 开始发更多老板本人的故事、价值观、对食材的坚持
   → 让老顾客不只是喜欢产品，而是喜欢"这家店的人"
```

## 二、预算策略分级方案

### 预算 A 级：0-500 元/月（零成本起步）

**策略核心：用时间换流量，只做能持续的事**

```
本月行动计划：
1. 认领高德/百度地图商家（第1周，1次完成）
2. 认领大众点评商家，上传 10 张产品照片（第1周）
3. 发 8 篇小红书（每周 2 篇，发 1 个月）
   - 内容主题：[地名]+产品笔记 × 4 + 制作过程 × 2 + 顾客推荐 × 2
4. 邀请 15 个来过的顾客在大众点评写评价
5. 建微信顾客群，发3条内容

月底检查：小红书哪篇互动最好？下个月继续做这种内容。
预计效果：建立基础搜索流量，开始有陌生顾客找上门
```

---

### 预算 B 级：500-2000 元/月（验证期）

**策略核心：小额测试，找到最有效的渠道，再加大投入**

```
预算分配建议：
- 内容制作（道具/美食摄影工具）：100-200 元
- 小红书薯条（给最好的 1-2 篇内容加热）：200-500 元
- 大众点评霸王餐（邀请 2-3 个小博主体验）：成本约 200-300 元
- 备用（如果薯条效果好，追加投入）：剩余

每笔钱投出去的判断标准：
- 薯条：1 篇内容花 100 元，带来了多少人私信/到店？
- 霸王餐：博主发了之后，搜索量有没有明显增加？
```

---

### 预算 C 级：2000-5000 元/月（放量期）

**策略核心：已验证有效的渠道，开始系统性投入**

```
预算分配建议（以 3000 元/月为例）：
- 小红书薯条：800 元（每周 200 元，选最好的内容加热）
- 微信朋友圈广告：1200 元（每天 40-60 元，定向周边 3 公里，跑 20-25 天）
- 大众点评活动/团购：500 元（设置 1-2 个引流团购，接受一定让利）
- 内容制作成本：300 元（道具/食材/摄影）
- 机动/测试：200 元

注意：这个预算级别，朋友圈广告的素材质量直接决定效果。
建议：不要用设计海报，用真实产品图 + 真实顾客场景图。
```

## 三、高效推广的通用策略原则

### 原则 1：内容必须比广告多
每发 1 条付费广告，配套发 3-5 条真实内容。
纯广告轰炸会让人反感，但真实内容能建立信任。

### 原则 2：先做免费，再做付费
在不知道"什么内容对这家店有效"之前，花钱是在烧钱。
先发 10-15 篇免费内容，看哪种类型有自然流量，再把钱押在那种类型上。

### 原则 3：一次只验证一件事
不要同时改三个变量（平台+内容+时间都换）。
每次只改一个，才知道是什么在起作用。

### 原则 4：把"第一次来"和"再次来"分开设计
不要用一个活动同时解决两个问题。
- 第一次来：需要"有理由来一趟"（优惠/好奇/朋友推荐）
- 再次来：需要"有理由回来"（情感连接/特权感/期待新产品）

### 原则 5：效果不好先看三个地方
1. 看到了但没来 → 转化问题，检查"来的理由"够不够充分
2. 看到了很多人但都不是目标客群 → 定向问题，平台或内容选错了
3. 来了一次但没再来 → 体验或复购问题，线下体验/后续跟进出了问题

## 四、内容创作快速公式

### 小红书笔记公式
```
标题（20字以内）：[地名/场景] + [产品名] + [情感钩子/具体描述]
示例：「光明社区藏着家烘焙店，每天限量50个的牛角包真的绝了」

封面图：产品特写（近景），背景简洁，有质感
第一张图最重要，决定有没有人点进来

正文结构：
第1段（2-3句）：场景代入，让目标顾客觉得"这说的是我"
第2段（3-5句）：产品核心卖点，具体到细节（用料/口感/做法）
第3段（1-2句）：地址 + 营业时间 + 行动号召

标签：#[城市][区域]美食 #[产品品类] #附近好店 #社区烘焙（每篇5-8个标签）
```

### 微信群消息公式
```
新品通知：
「今天新出了[产品]！[一句描述]，今天只做了[数量]份，先到先得。
想要的私信我，或者明天直接来拿。[附图]」

周末特惠：
「周六周日，买[A产品]送[小礼物]，限量[数量]份。不预约也可以来，但建议提前告诉我一声，给你留着。[附图]」

日常分享（不卖货）：
「今天早上做了一批[产品]，出炉的时候香气飘满整个走廊哈哈，
有个顾客还特地回头来多买了一份说是给朋友的。这种时刻还是很开心的。[附图]」
```

### 抖音本地推视频公式
```
前3秒（抓眼球）：最好看的动态画面——出炉/拉丝/切开/淋酱
中间10秒（说清楚）：产品名 + 今天有什么特别 + 价格/地址
最后3秒（要行动）：「[地名]的朋友，来找我」或「评论区见」

配文：[地名]附近的[产品]，每天限量，来晚了没有 📍[地址]
```
