## 小红书 — 社区烘焙/颜值类店铺首选

### 算法逻辑（用白话说）
小红书的内容能不能被人看到，取决于两件事：
1. **发出去的前2小时**，有多少认识你的人点赞收藏（冷启动）
2. **内容本身有没有搜索关键词**，让陌生人能搜到你

所以策略是：**内容能被搜到 + 发出去时有人帮你互动** = 流量进来。

### 适合社区小店的内容类型（按效果排序）

**① 地域+产品搜索笔记（效果最稳定）**
- 标题必须包含：地名 + 产品名 + 场景词
- 示例标题：「光明新区附近超好吃的软欧包！宝妈下午茶首选」
- 示例标题：「深圳光明 | 躲在社区里的烘焙小店，比网红店好吃10倍」
- 内容结构：产品特写图（3-5张）+ 一句核心卖点 + 地址/营业时间
- 发布时间：周四/周五下午2-4点（白领翻小红书高峰）

**② 制作过程/后厨日记（建立信任）**
- 拍面包/蛋糕的制作过程，展示用料干净、手艺用心
- 不需要专业设备，手机横拍就够
- 标题：「凌晨4点，我是怎么做一块吐司的」「为什么我的奶油用量是别人的2倍」

**③ 顾客故事/真实反馈（口碑内容）**
- 顾客发了好评或者拍了照，用她的图（经允许）做成笔记
- 标题：「附近的宝妈每周五都来，说是女儿最爱的生日蛋糕」

**④ 节日/季节内容（冲热搜）**
- 提前 10-15 天布局：母亲节蛋糕/中秋月饼/圣诞姜饼
- 标题要蹭节日搜索词：「2025母亲节蛋糕推荐丨光明本地自提」

### 关键数据参考（社区小店基准）
- 发出 10-15 篇后，开始出现稳定搜索流量（不要用 3 篇没效果就放弃）
- 每篇笔记从发出到被搜索到通常需要 3-7 天
- 一旦有 1 篇笔记跑起来，立刻发同类型的 2-3 篇跟进

### 小红书付费推广（蒲公英/薯条）
- **薯条**（给自己内容付费加热）：先发免费内容，选互动最好的那篇花 100-200 元薯条测试
- 不建议一开始就付费，先找到哪种内容有自然流量再放钱

## 微信生态 — 所有店铺的私域核心

### 微信的核心价值
微信不适合获取陌生新客，但它是**维系老顾客最高效的地方**。
一个来过的顾客进了你的微信群，比 10 个看过你小红书的陌生人更有价值。

### 微信私域搭建三步走

**第一步：建立顾客联系（进店时）**
- 顾客结账时说：「扫一下我这个码，加我微信，下次来提前告诉我，给你留新鲜的」
- 或者：「进我们的群，每周会发新品和只给老顾客的折扣」
- 不要说"加微信有优惠"，太功利，改说"进群提前知道新品/限量品"

**第二步：微信群运营节奏（每周固定动作）**
```
周一：发一条本周新品预告（图+简短文字）
周三：随手拍一条制作过程或者今天的新鲜出炉（建立日常感）
周五：发一条限量通知或者周末特惠（制造紧迫感）
节日前3天：发节日产品预告+预订方式
```
- 发的内容不要每次都是广告，要有生活感，让顾客觉得这个群"有意思"

**第三步：激活转介绍（老带新）**
- 发起活动：「把这条消息转发给一个朋友，来了凭截图送你一个小饼干」
- 或者：「你带一个新朋友来，你们俩各享88折」
- 简单的转介绍机制，不需要裂变工具，微信截图就够了

### 朋友圈广告（付费精准投放）
适用场景：已有一定产品口碑，想主动触达附近陌生人
- 最低预算：每天 50 元，跑 20 天 = 1000 元
- 定向设置：地理位置（店铺周边 3-5 公里）+ 年龄（25-40）
- 素材建议：真实产品图，不要设计感太强的海报（朋友圈里太广告感会被划走）
- 文案公式：「[地名]附近的[目标人群]，[产品]今天有[具体优惠/新品]，[行动号召]」

## 大众点评 / 美团 — 搜索意图最强的流量

### 为什么要做点评
来大众点评搜索"附近烘焙"的人，已经有明确购买意图了。这是质量最高的流量。
但这个平台是**被动等人来搜**，需要先积累评价和内容。

### 冷启动期（0-20条评价）的做法
1. **商家认领**：在大众点评/美团上认领自己的商家，完善菜单/环境照片/营业时间
2. **主动邀请真实顾客评价**：
   - 顾客买完之后说：「如果你觉得好吃，能在大众点评帮我写两句话吗？现在就几条评价，帮我攒一下」
   - 真诚请求，不要给钱或者让人刷评（会被封号）
3. **商家自己发图文**：点评支持商家自己发内容，定期更新产品图

### 积累到 20+ 条评价后的策略
- 开通「霸王餐」活动（免费给博主体验换评价）
- 设置团购券/优惠券（降低第一次来的门槛）
- 申请「必吃榜」提名（需要一定评价数量）

## 抖音本地推 — 适合有视频能力的店铺

### 适用条件（不是所有店都适合）
- 老板或员工能拍短视频（15-60秒）
- 产品有视觉冲击力（出炉、拉丝、切开等画面）
- 有时间每周至少发 3 条以上

### 本地推的逻辑
抖音的「本地推」功能可以把视频推给附近的人，不需要粉丝基础。
- 花 100 元，可以让你的视频推给大约 3000-5000 个附近的人
- 配合团购券，可以直接在视频下方放购买入口

### 视频公式（适合烘焙店）
```
开头（3秒）：最有冲击力的画面——出炉、拉丝、切开、奶油堆叠
中间（10秒）：价格/地址/今天的新品
结尾（3秒）：行动号召——「附近的朋友来找我，记得@一个想来的朋友」
```

## 高德/百度地图 — 零成本必做的基础动作

这是最被低估的免费流量。很多人找到一家店是因为在地图上搜索"附近烘焙"。

**必做的三件事（全部免费）：**
1. 认领高德/百度地图上的商家信息
2. 上传 5-10 张真实店面和产品照片
3. 完善营业时间/电话/地址（很多店这个都是错的）

做完这三步，附近用"导航找烘焙店"的人就能找到你。

## 平台组合推荐（按预算分级）

### 预算 500 元/月以下
```
主力：小红书（免费内容）+ 微信私域（建群，免费）
辅助：大众点评认领（免费）+ 地图认领（免费）
付费：不建议，先把免费渠道做扎实
```

### 预算 500-2000 元/月
```
主力：小红书（内容+少量薯条）+ 微信私域
付费试验：选小红书表现最好的 1-2 篇，花 200-300 元薯条加热
辅助：大众点评积累评价
```

### 预算 2000-5000 元/月
```
主力：小红书（内容+薯条）+ 微信朋友圈广告（定向周边）
辅助：大众点评（霸王餐/团购券）
可选：抖音本地推（如果能出视频）
```
